# Kyklos Roguelike

3D space roguelike based on the game Ringtaw (commonly referred to as just "marbles")

## Features
- Orbit-based movement
- Charge-shot combat
- Multiple projectile types
- Roguelike progression
- Upgrade/shop system

## Controls
- WASD — Move
- Mouse — Aim
- Left Click — Charge/Fire
- Q/E/etc — Swap ammo type

## Built With
- Godot 4
- GDScript
